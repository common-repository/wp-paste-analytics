Allows you to insert Google Analytics code to all active themes.

== Description ==

Allows you to insert Google Analytics code to all active themes. It is convenient if you frequently change the site template.

Features:

*   Automatic insertion of tracking code on Google or Yandex.
*   Supports the automatic insertion of a special meta tag that confirms the right of the site in search engines Yandex and Google.

Возможности:

*   Автоматически вставляет, указанный вами код счетчика Метрики/Google Analytics в любую активную тему.
*   Поддерживает автоматическую вставку специального мета-тега, подтверждающего права на сайт, в поисковых системах Яндекс и Google.

== Installation ==

1. Unzip files.
2. Upload 'shareurl' to the '/wp-content/plugins/' directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
